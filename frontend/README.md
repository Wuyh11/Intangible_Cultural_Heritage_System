# Frontend

## 技术栈

- Vue 3
- Vite 4
- TypeScript
- Pinia
- Vue Router
- Element Plus
- Axios
- ECharts

## 结构

- 公众门户：`/`、`/projects`、`/projects/:id`、`/map`、`/stats`
- 管理后台：`/admin/login`、`/admin/*`

## 环境变量

复制示例文件并按需调整：

- `.env.development.example`
- `.env.production.example`

## 前端特色

- 单前端双路由架构
- 本地湖北 GeoJSON 地图资源
- 共享请求封装与登录态守卫
- 同时支持门户展示与后台管理
