# 湖北省非遗信息管理系统

本仓库提供一个基于 `Spring Boot + Vue 3` 的湖北省非物质文化遗产信息管理系统本地代码版，包含：

- `backend/`：后端单体服务，负责登录鉴权、数据管理、统计分析、日志和导出
- `frontend/`：Vue 3 单页应用，同时包含公众门户和管理员后台
- `sql/`：MySQL 建表脚本、初始化数据和数据来源说明
- `deploy/`：服务器部署约定、环境变量和反向代理示例

## 当前阶段

本机阶段以“代码、SQL、配置和部署资产完整”为目标，不要求本机已接入 MySQL/Redis 运行。

## 默认演示账号

- 用户名：`admin`
- 密码：`Admin@123456`

## 主要能力

- 管理员登录与角色权限
- 非遗项目管理、图片维护、发布隐藏
- 传承人管理与项目关联
- 分类、地区基础数据管理
- 地图可视化、多维检索、数据统计
- 操作日志、Excel 导出
